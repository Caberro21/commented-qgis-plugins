Commented QGIS plugins

Here is a list of the plugins contained in this site, with a short description of their main features and their main areas of interest

  • what3words: This plugin uses a map tool to capture user interaction in the map canvas. It also features a Processing provider, so the capabilities of the plugin (converting coordinates into what3words adresses) are available as a Processing algorithm and can be used in the Processing elements (toolbox, modeler, batch interface, etc.)
  • MIL-STD-2525 symbolizer: This plugin allows to define symbology using the MIL-STD-2525 standard codes. It is a good example of how to define layer symbology programatically. It also shows how to use a custom renderer, and how to create a simple dialog for interacting with the user.
